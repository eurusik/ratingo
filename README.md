# Ratingo

**Ratingo** — україномовна платформа для відстеження трендів фільмів і серіалів. Проєкт агрегує рейтинги з TMDB, Trakt, IMDb та Metacritic, показує легальні платформи для перегляду та календар виходу епізодів.

Сайт: https://ratingo.top

---

## Призначення

Ratingo створений як зручний інструмент, який дозволяє:

- бачити актуальні тренди серед фільмів і серіалів;
- відстежувати зміну інтересу аудиторії з часом;
- швидко знаходити сервіси легального перегляду в різних регіонах;
- слідкувати за датами виходу нових епізодів.

Без новинного шуму, без піратських джерел, без зайвої маркетингової мішури.

---

## Поточний функціонал

- Трендові фільми та серіали за 30 і 90 днів.
- Агрегація рейтингів в одному місці (TMDB, Trakt, IMDb, Metacritic).
- Єдині картки контенту з описами, акторами, трейлерами та перекладами.
- Платформи легального перегляду для України та США.
- Календар виходу епізодів для серіалів.

---

## Підхід

Проєкт будується з фокусом на:

- україномовний інтерфейс;
- чистий, нейтральний дизайн без клікбейту;
- швидкість завантаження;
- коректні та перевірені дані;
- відсутність агресивної реклами.

Ratingo не намагається бути новинним ресурсом. Це радше інструмент для вибору, що дивитися далі.
