# Ratingo

Україномовна панель трендових серіалів і фільмів: рейтинги TMDB/Trakt/IMDb, провайдери перегляду та календар виходу епізодів (для серіалів).

**Для чого**

- Дізнатися, що зараз у тренді (серіали і фільми) та як змінюється інтерес глядачів.
- Подивитися де легально переглядати (по регіонах) і коли виходять нові епізоди (для серіалів).

**Можливості**

- Тренди за 30/90 днів з дельтами популярності для серіалів і фільмів.
- Єдині картки: переклади, рейтинги (TMDB/Trakt/IMDb/Metacritic), трейлери, актори.
- Провайдери перегляду (UA/US) і календар ефірів (для серіалів).

**Як працює наповнення**

- Серіали: короткі запити — координатор формує чергу, процесор обробляє задачі батчами; довгі етапи (backfill/meta/calendar) — окремо.
- Фільми: доступний повний синк трендів за один виклик.
- Деталі — `SYNC-GUIDE.md`.

**Стек**

- Next.js 16, React 19, Postgres + Drizzle ORM, Redis (опц.), k3s/ArgoCD або Vercel.

**Швидкий старт**

- Клон, `.env` на основі `.env.example`, `npm run dev`.
- Для наповнення бази — маршрути синку з `CRON_SECRET`:
  - Серіали: координатор/процесор/статус та повний синк (див. `SYNC-GUIDE.md`).
  - Фільми: повний синк трендів — `GET /api/sync/movies/trending/full`.

**Продакшн і CI**

- Деплой у k3s під ArgoCD; CronJobs для синку (оркестратор або окремі на етапи).
- GitHub Actions білдить і пушить Docker‑образ до GHCR.
