# Ratingo

Україномовна панель трендових серіалів: рейтинги TMDB/Trakt/IMDb, провайдери перегляду та календар виходу епізодів.

**Для чого**

- Дізнатися, що зараз у тренді та як змінюється інтерес глядачів.
- Подивитися де легально переглядати (по регіонах) і коли виходять нові епізоди.

**Можливості**

- Тренди за 30/90 днів з дельтами популярності.
- Єдині картки шоу: переклади, рейтинги (TMDB/Trakt/IMDb/Metacritic), трейлери, актори.
- Провайдери перегляду (UA/US) і календар ефірів.

**Як працює наповнення**

- Короткі запити: координатор формує чергу, процесор обробляє задачі батчами; довгі етапи (backfill/meta/calendar) — окремо.
- Деталі — `SYNC-GUIDE.md`.

**Стек**

- Next.js 16, React 19, Postgres + Drizzle ORM, Redis (опц.), k3s/ArgoCD або Vercel.

**Швидкий старт**

- Клон, `.env` на основі `.env.example`, `npm run dev`.
- Для наповнення бази — маршрути синку з `CRON_SECRET` (див. `docs/sync-trending.md`).

**Продакшн і CI**

- Деплой у k3s під ArgoCD; CronJobs для синку (оркестратор або окремі на етапи).
- GitHub Actions білдить і пушить Docker‑образ до GHCR.
